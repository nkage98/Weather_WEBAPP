# WeatherApp

WeatherApp project.
Backend API is responsible for authentication, weather data retrieval, and user favorites management.
The API acts as a middleware between the frontend and the weather provider, handling validation, error handling, and user-related operations.

## Tech Stack

### Backend
- Node.js
- Express
- TypeScript
- JWT Authentication
- Crypto
- REST API
- External Weather API (Open Meteo)

### Frontend

- React.js
- TypeScript
- Vite
- CSS
